# Sewon GitHub Profile README Starter

1. Create a public GitHub repository named exactly `andimsewon`.
2. Upload `README.md` to the repository root.
3. Upload the `assets/` folder as-is.
4. Commit changes.
5. Open `https://github.com/andimsewon` and check the profile README.

Optional terminal upload:

```bash
git init
git add README.md assets scripts
git commit -m "Create profile README"
git branch -M main
git remote add origin https://github.com/andimsewon/andimsewon.git
git push -u origin main
```
