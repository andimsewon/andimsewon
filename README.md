## What I work on

- **FAO Office of Innovation** — supporting knowledge discovery, AI enrichment, UX analysis, and data-visualization recommendations for the **ATIO Knowledge Base**.
- **Knowledge Graph + RAG** — building retrieval-augmented systems for multi-hop question answering with PyTorch and LangChain.
- **Computer Vision for Cultural Heritage** — led a DETR-based damage-inspection system with operational use, software copyright registration, and a patent filing.
- **IoT Field Systems** — co-authored **WaggleNet**, a LoRa/MQTT beehive-monitoring system validated in field deployment.
